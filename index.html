MANULÁ BABY

O Manulá Baby é um e-commerce desenvolvido do zero, desde a arquitetura do banco de dados relacional, até o layout das páginas, ele tem um sistema de administrador onde é possível cadastrar novos produtos, editar produtos postados, gerenciar os pedidos realizados e criar novas promoções, onde é possível inserir ou remover qualquer produto, dentro dessa promoção. O sistema de pagamento foi contruido em cima da API do Pagseguro, e os usuários que realizam as compras, podem ter o controle do carrinho de compras, status da compra e etc...

OSCAR JOIAS

O Oscar Joias é um e-commerce desenvolvido do zero, desde a arquitetura do banco de dados relacional, até o layout das páginas, ele tem um sistema de administrador onde é possível cadastrar novos produtos, editar produtos postados, gerenciar os pedidos realizados e modificar preços em massa por categoria, o adm pode selecionar a categoria desejada e incluir uma porcentagem a mais ou a menos em todos os produtos. ANOTAÇÕES BÍBLICAS

O Anotações bíblica foi um sistema web para leitura e estudo da bíblia, onde ele funcionava basicamente como uma rede social bíblica. Nele era possível realizar marcações e comentários em versículos desejados. Nos comentários o usuário podia escolher se queria deixa-lo como privado ou público, se ele escolhesse como público, todos que entrassem naquele versículo, poderiam ler o seu comentário. Também era possível seguir os usuários do sistema, para vizualizar os comentários daquela pessoa com mais facilidade.
